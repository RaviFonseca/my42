# Libft

## Description

Libft is a custom C library developed as part of the 42 curriculum.
The project consists of recreating several standard libc functions as well as additional utility functions.

---

## Compilation

Compile the library:

make

Clean object files:

make clean

Remove object files and library:

make fclean

Recompile everything:

make re
